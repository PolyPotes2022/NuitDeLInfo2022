# NuitDeLInfo2022

Projet réalisé par les PolyPotes pour la nuit de l'info 2022.

Le site est accessible à l'adresse suivante : http://polypotes.alwaysdata.net/

## Installation

```bash
git clone https://github.com/PolyPotes2022/NuitDeLInfo2022
cd NuitDeLInfo2022
npm i
npm run start
```
